## Описание
«Туристическое агентство Круиз» — это десктопное приложение для автоматизации работы туристического агентства, специализирующегося на продаже круизных путешествий, разработанное в рамках производственной практики на предприятии. Приложение позволяет пользователям просматривать каталог круизов, бронировать туры, управлять бронированиями и администрировать круизные предложения в удобной и интуитивно понятной форме.

## Установка
Убедитесь, что на компьютере установлена IntelliJ IDEA Community Edition 2024.2.2 (или другая среда разработки Java).

Скачайте файлы приложения:

CruiseApp.java

sqlite-jdbc.jar (драйвер базы данных)

Добавьте sqlite-jdbc.jar в зависимости вашего проекта:

File → Project Structure → Libraries → + → Java

Выберите файл sqlite-jdbc.jar

Скомпилируйте приложение CruiseApp.java

Запустите приложение

При первом запуске автоматически создастся файл базы данных cruise.db, а также:

будут созданы таблицы users, cruises, bookings;

добавлена учётная запись администратора по умолчанию:
Логин: admin
Пароль: admin123;

добавлены 6 тестовых круизов для демонстрации работы приложения.

## Использование
«Туристическое агентство Круиз» — это мощное и гибкое приложение для автоматизации деятельности туристического агентства, построенное на работе с локальной базой данных SQLite. Оно предоставляет удобный инструмент для:

- просмотра и фильтрации каталога круизов;

- бронирования круизов с расчётом итоговой стоимости;

- просмотра и отмены собственных бронирований;

- управления круизами (для администраторов);

- экспорта данных о бронированиях в CSV формат (для администраторов).

Приложение поддерживает два уровня доступа: клиент и администратор, что обеспечивает гибкое разграничение функциональных возможностей.
